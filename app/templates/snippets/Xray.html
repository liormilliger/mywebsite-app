JFrog Xray - Policy Rule Example

This is an example of a security policy rule in Xray to block artifacts with high-severity vulnerabilities.


{
  "name": "block-high-severity-vulns",
  "description": "Block download of artifacts with vulnerabilities rated High or above.",
  "type": "security",
  "rules": [
    {
      "name": "high-severity-rule",
      "criteria": {
        "min_severity": "High"
      },
      "actions": {
        "block_download": {
          "unscanned": false,
          "active": true
        },
        "fail_build": true
      }
    }
  ]
}
