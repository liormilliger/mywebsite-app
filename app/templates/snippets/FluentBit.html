Fluent Bit Configuration (fluent-bit.conf)

This config tails a log file and forwards new entries to Elasticsearch.

[SERVICE]
    Flush        1
    Daemon       Off
    Log_Level    info

[INPUT]
    Name         tail
    Path         /var/log/app.log
    Tag          app.log

[OUTPUT]
    Name         es
    Match        *
    Host         elasticsearch-master
    Port         9200
    Index        my-app-logs
    Type         _doc
